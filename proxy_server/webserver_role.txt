============================================================
 COMMON COMMANDS
============================================================
List vhosts:
- apachectl -S

Config test:
- sudo apachectl configtest

Reload Apache:
- sudo systemctl reload apache2

Full service status:
- systemctl status apache2 --no-pager

Tail logs:
- tail -f /var/log/apache2/error.log
- tail -f /var/log/apache2/*error.log

Check all sites:
- /opt/vhost-tools/check-all-vhosts.sh

Check one site:
- /opt/vhost-tools/check-one-vhost.sh 123hostedtools.com http://127.0.0.1

============================================================
 INTERPRETATION
============================================================
- backend fails = app or port problem
- http fails = vhost, rewrite, or port 80 problem
- https fails = cert, proxy, or port 443 problem
- local works, public fails = NAT / firewall / external path issue
- 403 may be GOOD for restricted apps like FreePBX
